# 灵虾使用手册

> 灵虾是你的专属 AI 助手。能聊天、查资料、写分析、做 PPT、跑代码，也能和同事助手协作，还能连微信随时找它。

---

## 快速上手

### 第一次用，三步找到手感

1. **先聊一句简单的话**：比如"你好，你能做什么？"
2. **再试一个真实任务**：比如"帮我看下今天 A 股收盘情况"
3. **最后再试复杂任务**：比如"做一份 AI 金融应用研究提纲"

### 领养你的灵虾

1. 登录灵感平台
2. 进入 **灵感养虾**
3. 领养后，系统会分配一个专属子域名
4. 打开专属页面即可开始对话

每只灵虾都是独立的：对话、记忆、技能、文件互不混用。

## 聊天页面

### 基本用法

聊天页面是和灵虾互动的主要入口。支持：

- **文字对话**：直接打字提问
- **语音输入**：点麦克风按钮说话，灵虾自动转文字后回答
- **模型切换**：顶部可切换不同模型（GLM-5、MiniMax M2.7、DeepSeek 等）
- **重置会话**：清空当前对话上下文，重新开始（不会删除记忆和文件）

### 流式输出

灵虾会一边生成一边往外吐字，不需要等整段内容全部完成。生成过程中可以点 **停止** 按钮中断。

### 工具调用

灵虾在回答过程中可能会调用工具（搜索、执行代码等）。你会看到：

- 工具调用卡片：显示工具名称、执行状态、耗时
- 产出文件卡片：工具执行完后如果生成了文件，会直接显示下载按钮

### 连接方式

灵虾支持两种连接方式，自动选择：

- **WebSocket（WSS）**：优先使用，实时双向通信
- **HTTP SSE**：WSS 不可用时自动降级（如企业内网环境），功能完全一致

在浏览器控制台可以看到 `[DIAG]` 开头的日志，显示当前走的是哪条路径。

## 微信集成

### 绑定微信

1. 进入侧边栏 **微信** 页面
2. 点击绑定，扫描二维码
3. 确认后即绑定成功

### 使用方式

绑定后，你可以直接在微信对话框里和灵虾聊天：

- **发文字**：直接发消息，灵虾会回复
- **发语音**：微信自动转文字后灵虾处理，效果和打字一样
- **解绑**：在微信页面点击解绑即可

灵虾收到微信消息后会调用主聊天的 AI 能力回复，和网页端用的是同一个大脑。

## 技能

### 技能是什么

技能是给灵虾安装的专门能力模块。比如查行情、读研报、生成 PPT、搜资讯等。

### 在哪里看

侧边栏 **技能** 页面。每个技能会显示：

- 来源：平台技能 / 私有技能 / 扩展技能
- 状态：已绑定 / 未绑定 / 需配置 / 缺依赖

### 技能市场

技能页面右上角可以进入 **技能市场**，浏览和安装更多技能。

### 为什么开了技能还不一定能用

两个状态要分清：

- **已安装**：技能文件存在
- **已绑定**：技能绑定到了你的 Agent，对话中可以调用

有些技能安装了但没绑定、或缺运行环境/配置，就会"看得见但用不了"。

## 协作广场

### 怎么打开

聊天页右上角的 **协作** 按钮，或侧边栏 **智能体协作**。

### 业务 Agent

协作面板里可以直接和专用业务 Agent 对话，比如：

- **PPT 生成助手**：描述主题、受众、页数，自动生成演示文档
- **代码助手**：在隔离沙箱里写代码、调试、运行
- **金融投顾**：DCF 模型、行业分析、研究结论整理
- **信用风险 / 债券分析**（远程 Agent）：对接专业后端系统

业务 Agent 的会话独立管理，关闭面板后短时间内重新打开可继续。

### 跨助手协作（重点功能）

灵虾支持两种协作模式：

#### 模式一：Agent 之间协作

你的灵虾可以调用同事的灵虾来完成任务。比如：

1. 你需要一份信贷分析报告，但你的灵虾没有信贷分析能力
2. 同事的灵虾配置了「灵犀 · 智贷决策助手」技能
3. 你在协作广场找到同事的灵虾 → 发起协作请求
4. 同事审批通过 → 他的灵虾自动执行分析 → 结果返回给你

#### 模式二：人与人通过 Agent 协作

团队成员可以互相「借用」对方灵虾的能力：

1. 打开侧边栏「智能体协作」
2. 在「协作目录」里看到组织内其他成员的灵虾
3. 点击对方灵虾 → 发起协作 → 选择任务类型和描述
4. 对方收到请求 → 审批（或自动通过）→ 执行 → 结果回传

#### 协作设置

在「设置」标签页配置你的灵虾如何接受协作：

- **可见性**：私有 / 组织内可见 / 全平台可见
- **接受聊天**：是否允许别人跟你的灵虾聊天
- **接受任务**：不接受 / 需要审批（推荐）/ 自动接受
- **任务白名单**：允许哪些类型的任务自动通过

#### 协作状态

- **收到的请求**：别人调用你的灵虾时，可以批准/拒绝/查看执行过程和结果
- **我发出的请求**：你调用别人灵虾时，实时查看状态（待审批/执行中/已完成）

## 定时任务

侧边栏 **定时任务** 页面。可以设置灵虾定期自动执行任务：

- **每隔 N 分钟/小时**执行
- **每天固定时间**执行
- **Cron 表达式**精确控制

支持的任务类型：

- 发一条消息让灵虾处理（agentTurn）
- 系统事件触发（systemEvent）

每次执行的结果（摘要、token 用量、耗时）都可以在历史记录里查看。

## 安全边界

### 灵虾能做什么

在授权范围内：聊天、调用技能、生成文件、执行代码（沙箱隔离）、协作。

### 灵虾不能做什么

- 读取别人的文件
- 获取系统密钥
- 越过隔离边界
- 修改系统级配置

### 代码执行安全

灵虾的代码执行在隔离沙箱中运行：

- 无网络访问
- 只读根文件系统
- CPU、内存、进程数、超时均有限制
- 5 分钟执行超时自动中断

## 常见问题

### 技能开了但感觉没效果？

1. 检查技能是否"已绑定"（不只是已安装）
2. 重置会话后再试

### PPT 生成失败怎么办？

把需求说得更具体：主题、受众、页数、风格偏好。越具体成功率越高。

### 想让它记住某件事？

直接说"记住xxx"或"以后按xxx方式回答我"。

### 微信发语音没反应？

确认已绑定微信。语音消息由微信自动转文字后处理，和打字效果一样。

### 在企业内网用很慢或消息结束不了？

企业网络可能屏蔽 WebSocket，灵虾会自动降级到 HTTP。如果遇到异常，尝试刷新页面重试。

### 会话到期了数据还在吗？

- 对话按会话生命周期管理
- 记忆文件长期保留
- 已生成的文件通常也还在

## 金融场景实操指南

以下是几个典型的金融业务场景，一步一步教你怎么用。

### 场景一：晨间市场速览

> 目标：每天开盘前自动查股价，结果推送到你的微信。

**第一步：绑定微信**

1. 打开灵虾网页，点击左侧导航栏「微信」
2. 点击「扫码绑定微信」按钮
3. 打开手机微信，扫描屏幕上的二维码
4. 在微信中确认绑定
5. 页面显示"已绑定"就成功了

**第二步：创建定时任务**

在主聊天页面的输入框里，直接打字：

```
每天早上9点25查一下工商银行和建设银行的股价，发到我微信
```

按回车发送。灵虾会自动理解你的意思，然后显示：

```
✅ 定时任务已创建

| 项目     | 内容                           |
|----------|-------------------------------|
| 任务名称 | 工商银行建设银行股价             |
| 执行内容 | 查询工商银行和建设银行的股价      |
| cron     | 25 9 * * *                     |
| 推送渠道 | 微信                           |
```

**第三步：等着收微信就行**

每天早上 9:25，灵虾会自动查询股价，然后把结果推送到你的微信。你不需要打开电脑，手机上就能看到。

**管理你的定时任务**

- 想看有哪些任务 → 在聊天里说"我有哪些定时任务"
- 想删除某个任务 → 在聊天里说"删除工商银行那个定时任务"
- 想修改 → 先删除再重新创建一个
- 也可以在左侧导航栏「定时任务」页面里直接管理

### 场景二：个股快速分析

> 目标：对一只股票做快速分析，了解当前走势和投资信号。

**第一步：打开股票分析**

1. 点击聊天页面右上角的「⚡ 协作」按钮
2. 在右侧面板里，找到「灵犀 · 股票分析」
3. 点击打开

**第二步：输入分析请求**

```
分析工商银行（601398）最近走势
```

或更具体的：

```
比较招商银行和兴业银行的技术面指标
```

**第三步：查看分析结果**

系统会分析并返回：

- 当前价格和涨跌幅
- 技术指标（MACD、RSI、布林带等）
- 资金流向和筹码分布
- 近期消息面汇总
- 交易信号建议

支持 A 股、港股、美股，11+ 种交易策略。

### 场景三：信贷风控辅助

> 目标：对一笔贷款申请做快速风险评估。

**第一步：打开智贷决策助手**

1. 点击右上角「⚡ 协作」按钮
2. 找到「灵犀 · 智贷决策助手」
3. 点击打开

**第二步：输入贷款信息**

把你掌握的借款人信息整理好，输入到对话框。越详细越好：

```
借款人：某制造业企业
申请：流动资金贷款 3000 万
近三年营收：2亿、1.8亿、2.2亿
资产负债率：68%
主要客户：某汽车集团（占营收60%）
担保方式：厂房抵押
请做信用风险初评
```

**第三步：查看风险评估结果**

助手会给出结构化的评估报告：

- **风险等级**：低 / 中 / 高
- **财务分析**：营收趋势、负债水平、现金流评价
- **行业风险**：所在行业的景气度和风险提示
- **关注点**：客户集中度过高（60%）、资产负债率偏高等
- **尽调重点**：建议实地核查的内容
- **授信条件建议**：额度、期限、利率区间、增信措施

**第四步：继续追问细节**

你可以在同一个对话里继续追问：

```
如果资产负债率降到60%以下，风险等级会变吗？
```

```
这个行业最近有什么监管政策变化？
```

助手会基于上下文继续分析，不需要重新输入借款人信息。

### 场景四：研究报告快速生成

> 目标：把一个分析主题做成可汇报的 PPT。

**第一步：在主聊天里说需求**

直接在主聊天输入框输入：

```
帮我做个PPT，主题是AI在银行业的应用趋势
```

**第二步：选择使用专业助手**

灵虾会弹出一张推荐卡片：

```
💡 检测到专业需求，推荐使用：
📊 灵匠 · 幻灯片（PPT）

[👉 打开助手]    [💬 继续对话]
```

- 点击「打开助手」→ 右侧弹出 PPT 助手面板
- 点击「继续对话」→ 不用专业助手，让主聊天 AI 直接回答

**第三步：在 PPT 助手里细化需求**

点击「打开助手」后，在右侧面板里补充细节：

```
做8页，给行长汇报用，偏战略分析风格。
重点包括：国内外银行AI应用现状、核心场景、风险挑战、未来趋势。
```

**第四步：等待生成并下载**

- 助手会自动搜索资料、组织内容、生成 PPT
- 完成后会出现下载按钮，点击即可下载 .pptx 文件
- 如果不满意，可以继续对话修改："第三页内容太长，精简一下"

> 提示：如果想要网页版演示文稿（带动画效果），可以选择「灵匠 · 幻灯片（HTML）」助手。

### 场景五：跨领域合规问答

> 目标：一个问题涉及多个金融领域（信贷+保险+风控），需要综合回答。

**第一步：打开共享智脑**

1. 点击右上角「⚡ 协作」按钮
2. 找到「灵枢 · 共享智脑」
3. 点击打开

**第二步：提出跨领域问题**

```
一家小微企业要贷款500万买厂房，需要什么保险保障？贷款审批要注意什么风险点？
```

**第三步：获得综合方案**

共享智脑会从多个专业角度同时回答：

**信贷视角**：
- 厂房作为抵押物的评估要求
- 小微企业授信的特殊政策
- 还款能力分析要点

**保险视角**：
- 抵押物保险（财产一切险）
- 企业责任险配置建议
- 保费估算参考

**风控视角**：
- 行业风险评级
- 抵押物价值波动风险
- 贷后管理建议

你可以继续追问任何一个方向的细节，智脑会保持跨领域的视角。

### 场景六：数据处理与分析

> 目标：用代码处理数据、生成图表。

**第一步：在主聊天里说需求**

```
帮我写个Python脚本，生成一个柱状图
```

灵虾会推荐使用代码助手。点击「打开助手」。

**第二步：在代码助手里描述具体需求**

```
用 matplotlib 画一个柱状图，横轴是2020到2025年，纵轴是营收（亿元），
数据是：18, 20, 19, 22, 25, 28。标题写"某企业近六年营收趋势"。
保存为 /output/revenue.png
```

**第三步：查看结果**

- 代码在安全沙箱里运行（不会影响系统）
- 执行完成后，会出现文件下载卡片
- 点击下载即可获得图表图片

**第四步：调整优化**

不满意可以继续说：

```
柱子颜色改成蓝色渐变，加上每个柱子顶部的数字标签
```

助手会修改代码重新运行，你再下载新版本即可。

### 场景七：微信随时问

> 目标：不在电脑前时，用微信和灵虾交互。

**前提：先绑定微信**（参见场景一的第一步）

**用法一：文字提问**

在微信里直接给灵虾发消息：

```
今天工行股价多少
```

几秒后灵虾回复最新行情数据。

**用法二：语音提问**

按住微信的语音按钮，说：

```
"帮我查一下最近有什么金融科技方面的政策"
```

灵虾会自动把语音转成文字，然后回复分析结果。跟打字效果一样。

**用法三：接收推送**

如果你设置了定时任务（场景一），灵虾会主动给你发微信：

```
⏰ 定时任务「工行股价」

📈 工商银行（601398）最新行情
最新价：6.82  涨跌幅：+0.44%
成交量：3.2亿手  成交额：21.8亿
```

你不需要做任何操作，到点就能在微信里看到。

> 注意：微信端和网页端用的是同一个 AI 大脑，对话记忆互通。微信里问过的问题，回到网页上也能看到上下文。

灵虾不只是一个聊天框。日常用它聊天查资料，复杂任务交给专用 Agent，长期偏好交给记忆，协作任务放进协作广场，微信随时找它——这样最顺手。
